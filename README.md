Music Player

A simple web-based music player built with HTML, CSS & JavaScript. This project is a lightweight music player you can open in any modern browser. It comes bundled with a few songs (MP3 files) and lets you play/pause and switch tracks using a custom interface.

## 🚀 Features

- Clean and responsive UI (desktop & mobile)  
- Play / Pause controls  
- Next & Previous track support  
- Built with pure HTML, CSS, and JavaScript  
- Standalone — no backend or server required

## 🧠 How It Works

1. Open `index.html` in your browser.  
2. The playlist is powered by JavaScript — it loads the MP3 files included in the repo.  
3. Controls let you play, pause, and switch between audio tracks.  
4. Additional styling in `style.css` enhances the UI while `script.js` handles playback logic.

## 🗂️ Repository Structure

Music-Player/  
├── index.html         ← Main player page  
├── style.css          ← Player styling  
├── script.js          ← JS for player controls  
├── *.mp3              ← Sample music files  
└── (other assets)     ← Icons, album art, etc.

## 📦 Usage

1. Clone the repository:

```
git clone https://github.com/Deepsgit2025/Music-Player.git
```

2. Open `index.html` in any browser.  
3. Start playing music instantly.

## 🙌 Customization

You can easily add your own songs or customize the playlist:

- Add new MP3s to the repository.  
- Update the `trackList` array in `script.js` with the new file names.  
- Modify styles in `style.css` to change look & feel.

## 📍 Technologies

- HTML5 Audio API  
- JavaScript  
- CSS3

## 📝 Notes

- Currently this is a static player that works entirely in the browser.  
- No backend, database, or server technologies are required.
